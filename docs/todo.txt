
search
======
SQL generation
get_nodes([<name1, value1>, ...])
get_links([<name1, value1 >, ...])

cmd
===
load
cd: change absolute if starts with /, otherwise relative to current
dir: list sub-dirs and files of the given node

setprop path name value
delprop path name

link fromnodepath tonodepath linktype
unlink fromnodepath tonodepath linktype

setlinkprop path name value
dellinkprop path name

tell: dump everything about the given node: path, props, in-links, out-links
